px UNIT



/* px is standard unit */

% UNIT

/* percentage (%) is the relative unit and it will be relative to its parent element for example section is parent of sectiondiv and sectiondiv is parent of div and body tag is parent of section and html tag is the parent of body tag. */

em UNIT

x em means x times of the size of the font in the nearest parent element. Like by default font size is 16px and write like 2rem then the size will became two times of the 16px that is 32px.

rem UNIT

this quantity is relative to font size of the root element i.e html tag

vw and vh UNIT

x vw means x% of the width of the viewport. And viewport means the area visible to the user on the web, this emplies that on zooming also the div will take the 50% of that viewport. And vh is also the same thing but only for height of the div.
